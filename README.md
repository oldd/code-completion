# Code Completion

In these exercises you will be given incomplete snippets of code and their expected behavior.  Your task is to fill in the blanks so the code does what it is supposed to.   Anything that passes the tests is a correct solution.

## Specifications

0. Study the [completed examples](./examples-to-study) to see how to use these exercises.
1. Complete the [provided exercises](./exercises-to-complete).  
  * Copy paste the provided (incomplete) code into the developer console in your browser and run it, modifying the missing lines, until you pass all of the tests. 
  * There are no tricks in these challenges, the code provided works and is simply missing some steps.  
2. Design your own challenges for others to complete in the ["my-challenges"](./my-challenges) folder.  You can build your challenges from examples in your favorite tutorials, from code you've already stepped through, or from thin air. 
3. Assign your challenges to other students with GitHub Issues, and complete other people's challenges with Pull Requests.
